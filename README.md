# The Rewind

## Description
**The Rewind** is a browser-based "Finish the Lyrics" game built with HTML, CSS, and vanilla JavaScript. Designed with a custom Philadelphia sunset aesthetic, the game challenges players to test their music knowledge across multiple eras. Players choose a genre, enter their name, and race to answer lyric questions before the 15-second countdown runs out.

---

## Features
* **Philly Sunset Interface:** A custom light-mode UI featuring a skyline silhouette and city-glow styling.
* **Curated Genres:**
    * 2000s R&B Hits
    * 2010s Popular Hits
    * **Beyoncé Discography:** A deep library spanning her entire solo catalog.
* **Multiple Game Modes:**
    * **Single Player:** Test your own personal high score.
    * **Host Game Room:** Generate a unique room code for group sessions.
* **Interactive Elements:**
    * 15-second countdown "sprint" for every question.
    * Live score and streak tracking.
    * Artist credits displayed with every song title.
    * Personalized leaderboard entries.

---

## How to Play
1. Open `index.html` in any modern web browser.
2. Enter your player name.
3. Choose either **Single Player** or **Host Game Room**.
4. Select a genre and click **Start Round**.
5. Answer the missing lyric before the 15-second timer reaches zero.
6. Check the leaderboard at the end of the round to see your standing.

---

## Requirements
* A modern web browser (Chrome, Safari, Edge, or Firefox).
* No installation or local server required.
